# Evidencias de tests automatizados

Esta carpeta contiene evidencias relacionadas con la ejecución de pruebas automatizadas del sistema de reserva de canchas de tenis.

## Objetivo

El objetivo de estas evidencias es demostrar que se ejecutaron pruebas unitarias e integración sobre las funcionalidades principales del sistema.

## Tipos de pruebas ejecutadas

Se ejecutaron:

- Pruebas de integración de usuarios.
- Pruebas de integración de canchas.
- Pruebas de integración de reservas.
- Pruebas de integración de reportes.
- Pruebas unitarias de reservas, pagos, horarios, cálculo de importes y superposición.

## Comando de ejecución general

```bash
python -m pytest
```
